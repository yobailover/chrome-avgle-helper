## Download dependencis into this directory

下载相关依赖到这个目录. (装完依赖后, 目录结构会长的像下面这个样)

After you download dependencies into this directory, this directory will looks like this:

```
- windows-libs
|-- ffmpeg-4.0.2-win64-static
	|-- bin
	|-- doc
	|-- presets
|-- wget-1.19.4-win64
```
